# Education Portal Node.js Project

## Start Project
- Install Node.js
- Install MongoDB

```bash
$ npm install
$ npm start
```

## Used Technologies
- Node.js
- Express.js
- express-session
- express-validator
- MongoDB
- Mongoose
- EJS
- Bootstrap
- Postman & API
- Middleware
- bcrypt
- connect-flash
- nodemailer
- slugify
